# A simple bookstore app 

> Built with Spring + PHP (a very unusual stack)


# Instalation

Required:
- PHP 8+
- Maven
- php-curl

First serve your api 
- From /backend
mvn spring:boot -run

Then serve your pages
- From /frontend
php -S localhost:8000
